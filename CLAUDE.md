# CLAUDE.md

This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## Project

PSIops (Photometrically accurate Science Image Operations) — a Python library for scientific image processing built on NumPy and SciPy.

## Setup

```bash
pip install -e ".[dev]"
```

## Running checks

Run everything (parallel by default):

```bash
scripts/run-all-checks.sh
```

Targeted checks:

```bash
scripts/run-all-checks.sh --ruff-check      # lint only
scripts/run-all-checks.sh --mypy            # type check only
scripts/run-all-checks.sh --pytest          # tests only
scripts/run-all-checks.sh -d                # docs and Markdown only
```

## Code style (differs from defaults)

- **Line limit: 90 characters** (not PEP 8's 79)
- **Single quotes** for strings
- Do not run `ruff format` — format code manually to match style
- Type hints required on all function parameters and return values
- Google-style docstrings

## Testing

- Tests run in parallel by default (`-n auto`); use `scripts/run-all-checks.sh -w 1` to run serially
- All runtime warnings are elevated to test failures (`filterwarnings = ["error"]`)
- Coverage minimum is 90%; tests fail below this threshold
- `MYPYPATH=src` is required for mypy: `MYPYPATH=src python -m mypy src tests`

## Source layout

- Package source: `src/psiops/`
- Tests: `tests/`
- `src/psiops/_version.py` is auto-generated by setuptools_scm — do not edit it manually
